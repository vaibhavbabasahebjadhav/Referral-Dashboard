# Referral Dashboard

A secure Referral Dashboard application built using React and Vite.

## Features

- User Authentication
- JWT Token Management
- Protected Routes
- Dashboard Analytics
- Referral Tracking
- Referral Details Page
- Logout Functionality
- API Integration

## Technologies Used

- React.js
- Vite
- React Router DOM
- JavaScript
- CSS3
- js-cookie

## Installation

```bash
npm install
npm run dev
